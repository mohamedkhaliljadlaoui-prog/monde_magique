# 🌍 Monde Magique - Voyage Éducatif Interactif pour Enfants

## 📝 Description du Projet

**Monde Magique** est une expérience éducative immersive et ludique qui transforme l'apprentissage de la géographie en une aventure extraordinaire. À bord d'un **train magique animé**, les enfants parcourent **10 destinations mondiales** à travers des cartes interactives colorées au **style cartoon vibrant** avec chaque pays illustré dans des couleurs vives et distinctes.

## ✨ Fonctionnalités Actuelles
