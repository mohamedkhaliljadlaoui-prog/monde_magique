🔧 Bouton "Réessayer" - Détails

Quand apparaît-il? Après une réponse incorrecte au QCM

Apparence: 🔄 أعد المحاولة

Fonction:

  • Réinitialise la question
  • Efface la réponse précédente
  • Permet de réessayer illimité
  • N'affecte pas les autres questions

Code:

function retryQuestion(q){ // Réinitialise la question complètement const c = document.querySelector(`[data-question="${q}"]`); // Enlever les classes de correction c.classList.remove('correct','incorrect'); // Réinitialiser les boutons c.querySelectorAll('.option-btn').forEach(b => { b.classList.remove('selected','disabled',...); }); // Réactiver le bouton vérifier c.querySelector('.verify-btn').disabled = false; }
