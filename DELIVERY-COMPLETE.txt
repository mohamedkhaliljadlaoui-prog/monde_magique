📞 SUPPORT & TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

If test-database.php FAILS:
  1. Open XAMPP Control Panel
  2. Verify Apache = "Running" (green)
  3. Verify MySQL = "Running" (green)
  4. Click "Start" if not running
  5. Refresh page (F5)

If login FAILS:
  1. Check console (F12 → Console tab)
  2. Look for red error messages
  3. Verify email/username not taken
  4. Try different credentials

If data NOT SAVING:
  1. Open test-database.php
  2. Verify "users" table exists
  3. Verify "progress" table exists
  4. Check phpMyAdmin: localhost/phpmyadmin/
  5. Look for database "mondo_magique"
