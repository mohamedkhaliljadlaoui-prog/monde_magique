# 🎮 Système de Progression des 10 Stages - Guide Complet

## ✨ Fonctionnalités Activées

### 1️⃣ **Déblocage Progressif**
- **Stage 1** : Toujours accessible 🔓
- **Stages 2-10** : Se déverrouillent automatiquement après avoir complété le stage précédent
- Le stage suivant ne peut être acces **que si** le score QCM ≥ 80% au stage précédent

### 2️⃣ **QCM Spécifiques pour Chaque Stage**

#### Stage 1: المجال الجغرافي
1. ما هو المجال الجغرافي؟ → **ب) رقعة محددة لها حدود وخصائص**
2. ما الذي يشمله المجال الطبيعي؟ → **ب) الصخور والهواء والماء**
3. ماذا يمثل المجال الجغرافي اجتماعياً؟ → **ب) العلاقات الاجتماعية والقيم**
4. ما هو المجال الجوي؟ → **ب) الفضاء الجوي**
5. ما هو المجال البحري؟ → **ج) أجزاء البحار**

#### Stage 2: المغرب العربي
1. أين يقع المغرب العربي؟ → **ب) شمال القارة الإفريقية**
2. ما هو أكبر بلد من حيث المساحة؟ → **ج) الجزائر**
3. كم عدد دول المغرب العربي؟ → **ج) 5 دول**
4. ماذا يغطي أغلب المساحة؟ → **ج) الصحراء**
5. كيف يكون المناخ في الجنوب؟ → **ج) جاف**

#### Stage 3: البلاد التونسية
1. أين تقع تونس؟ → **ب) شمال قارة إفريقيا**
2. ماذا يحدّ تونس من الشمال والشرق؟ → **ب) البحر الأبيض المتوسط**
3. إلى كم منطقة ينقسم المجال الجغرافي؟ → **ب) ثلاث مناطق**
4. كيف يكون المناخ في الجنوب التونسي؟ → **ج) صحراوي حار وجاف**
5. أيّ من الموارد الطبيعية؟ → **ب) الحبوب والزيتون والتمور**

#### Stage 4: المشهد الريفي المحلي
1. ما المقصود بالمشهد الريفي؟ → **أ) صورة وقتية لمجال هيأه الإنسان**
2. ينقسم المشهد الريفي إلى؟ → **ب) مسيّج ومفتوح**
3. ما الذي يميز المشهد الريفي المسيّج؟ → **ب) أراضيه محدودة بسياج**
4. أيّ من العناصر من المجال الزراعي؟ → **ب) الحقول والبساتين**
5. من أنواع السكن الريفي؟ → **أ) مساكن متفرقة ومساكن متجمعة**

#### Stage 5: المشهد الحضري
1. ما هو المشهد الحضري؟ → **ب) صورة لمجال هيّأه الإنسان**
2. ماذا يميّز المباني في المدن؟ → **ب) البناء العمودي**
3. كيف تكون المساكن في أطراف المدينة؟ → **ب) مساكن فردية قليلة الارتفاع**
4. ماذا يميّز المدينة؟ → **ج) كثرة الخدمات والمعالم**
5. كيف تكون الكثافة السكانية؟ → **ج) مرتفعة جدًا**

#### Stage 6: العلاقة بين الريف والمدينة
1. ماذا يُميّز الريف؟ → **ب) قلة السكان واعتمادهم على الفلاحة**
2. ماذا تُميّز المدينة؟ → **ج) كثرة السكان وتنوّع الأنشطة**
3. ماذا يقدّم الريف للمدينة؟ → **ب) المواد الغذائية**
4. كيف يتمّ الربط بين الريف والمدينة؟ → **ج) عبر الطرقات ووسائل النقل**
5. ما أهمية العلاقة؟ → **أ) خلق التوازن وتلبية الحاجيات**

#### Stage 7: توزع السكان في تونس
1. أين تتركّز الكثافة السكانية المرتفعة جدّا؟ → **ب) تونس الكبرى**
2. كيف تكون الكثافة السكانية في السواحل؟ → **أ) مرتفعة**
3. ما سبب ضعف الكثافة في المناطق الداخلية؟ → **ج) صعوبة المناخ وقلّة الأمطار**
4. أين تتركّز الكثافة الضعيفة جدا؟ → **ج) الجنوب التونسي**
5. ما معنى "الإقليم"؟ → **ب) جزء من التراب التونسي حسب الاتجاهات**

#### Stage 8: التوزّع الفلاحي
1. ماذا يُعتبر النشاط الفلاحي في تونس؟ → **ب) نشاط أساسي**
2. أين يتركّز الإنتاج الفلاحي الرئيسي؟ → **ب) الشمال والوسط**
3. ما هي المحاصيل الرئيسية؟ → **ب) الحبوب والزيتون والتمور**
4. ما أهمية الفلاحة في الاقتصاد؟ → **ب) أهمية كبيرة**
5. ماذا يحدّ النشاط الفلاحي؟ → **ج) ندرة الماء والمناخ الجاف**

#### Stage 9: الصناعة في تونس
1. ماذا يميّز الموارد الطاقيّة والمنجميّة؟ → **ب) قليلة وموزّعة على كامل البلاد**
2. في أيّ جهة تكثر مناجم الفسفاط؟ → **ج) الجنوب الغربي**
3. أين تتركّز أغلب الصناعات؟ → **ب) في السواحل**
4. أيّ من الصناعات شهدت تطوّرًا؟ → **ب) صناعة النسيج**
5. لماذا يجب ترشيد استهلاك الموارد؟ → **ج) لأنها قليلة وفي طور النضوب**

#### Stage 10: التجارة الخارجية التونسية
1. على ماذا تعتمد الصادرات أساسًا؟ → **ب) المنتجات الصناعية**
2. نحو أيّ جهة تتجه أغلب الصادرات؟ → **ب) أوروبا**
3. ممّ تتكوّن الواردات التونسية؟ → **أ) مواد صناعية وغذائية وطاقية**
4. ماذا تعاني منه تونس؟ → **ج) عجز تجاري متزايد**
5. ما هو أهمّ شريك تجاري؟ → **أ) الاتحاد الأوروبي**

### 3️⃣ **Système de Récompenses**
- 💎 **Diamants** : +10 par réponse correcte au QCM
- 🪙 **Coins** : +25 par réponse correcte
- **Bonus Essai** : Points calculés sur la longueur (min 50%, max 100% based on word count)

### 4️⃣ **Architecture de Navigation**

```
Dashboard (dashboard-stages.html)
    ↓
Stage 1 (80% QCM) ✓ → déverrouille Stage 2
    ↓
Stage 2 (80% QCM) ✓ → déverrouille Stage 3
    ↓
Stage 3 (80% QCM) ✓ → déverrouille Stage 4
    ↓
... (mêmes règles pour stages 5-9)
    ↓
Stage 10 (80% QCM) ✓ → retour au Dashboard
```

## 🚀 Comment Démarrer

### Accès au Système:
```
1. Ouvrir le navigateur
2. Aller à: http://localhost/monde-magique/dashboard-stages.html
```

### First Time Users:
- Tous les stages sauf le 1er sont verrouillés 🔒
- Cliquer sur **Stage 1** pour commencer

### Progression:
- Chaque stage a 6 étapes:
  1. 🖼️ **Début** - aperçu introductif
  2. 🎨 **Image 1** - première illustration
  3. 📸 **Image 2** - deuxième illustration
  4. 📚 **Leçon** - lecture du PDF
  5. ❓ **QCM** - 5 questions (80% requis)
  6. ✍️ **Essai** - rédaction personnelle

### Déblocage:
- Après une réponse QCM correcte → message ✓ vert
- Score < 80% → message ⚠️ rouge "Réessayer"
- Score ≥ 80% → bouton "Étape suivante" s'active
- À la fin: bouton vers stage suivant ou dashboard

## 🐛 Troubleshooting

**Stages verrouillés après rafraîchissement?**
- Vérifier que localStorage n'est pas désactivé
- Utiliser Dev Tools (F12) → Application → localStorage

**QCM ne valide pas?**
- Vérifier que les 5 questions sont répondues
- Score s'affiche en % après vérification

**Session perdue?**
- Données stockées dans localStorage (ne pas effacer)
- Cache navigateur ne devrait pas les affecter
