### Option 2: VERSION ULTRA-RAPIDE (avec Groq - 5 minutes)

Pour avoir un chatbot **AUSSI RAPIDE que ChatGPT**:

#### Étape 1: Obtenir clé Groq (2 min)
1. Allez sur: **https://console.groq.com**
2. Créez un compte (gratuit)
3. Cliquez sur "API Keys"
4. Créez une clé (format: `gsk_...`)
5. Copiez-la

#### Étape 2: Coller la clé (1 min)

**Dans `chatbot-super-intelligent.html` ligne 425:**
```javascript
key: 'gsk_VOTRE_CLE_ICI', // ← Collez votre clé ici
```

**Dans `php/api/chatbot-advanced.php` ligne 17:**
```php
'key' => 'gsk_VOTRE_CLE_ICI', // ← Collez votre clé ici
```

#### Étape 3: Tester!
```
Ouvrez: http://localhost/monde-magique/chatbot-super-intelligent.html
```

**Maintenant vous avez un chatbot ultra-rapide avec Llama 3 70B!** ⚡
